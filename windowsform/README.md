# Employee Management System – WinForms

A Windows Forms application built using C# and SQL Server.

## Features
- CRUD operations using Stored Procedures
- Employee photo upload (VARBINARY)
- Windows Authentication
- ADO.NET integration

## Tech Stack
- C#
- WinForms
- SQL Server
- Stored Procedures

## How to Run
1. Execute SQL scripts in dbo.Procedure.sql
2. Update connection string in App.config
3. Run the application
