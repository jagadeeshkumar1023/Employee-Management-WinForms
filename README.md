# Employee Management System (WinForms)

A Windows Forms application built using C# and SQL Server.

## Features
- Add, update, delete employees
- Store employee photos
- Uses SQL Server Stored Procedures
- Windows Authentication

## Technologies Used
- C# (.NET WinForms)
- SQL Server
- ADO.NET
- Stored Procedures

## Database Setup
- Run `dbo.Procedure.sql` to create required stored procedures
- Update connection string in `App.config`

## Screens
- Employee Details Form (CRUD operations)
